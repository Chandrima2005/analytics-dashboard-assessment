Welcome to Dashboard

The EV Dashboard website provides a clean and interactive interface to visualize key insights about electric vehicle (EV) data. It showcases data such as vehicle count, state-wise distribution, growth trends, and EV adoption over time. The dashboard includes informative charts and maps for better understanding of EV population dynamics. It is designed to be user-friendly and responsive, making data exploration intuitive. This tool is ideal for researchers, policymakers, and enthusiasts interested in tracking India's EV ecosystem.
